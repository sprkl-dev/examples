# How to run?

## Running regularly

```bash
node index.js
```

## Running with Sprkl
Before running, checkout the `index.js` file, uncomment line #5.
https://github.com/sprkl-dev/examples/blob/main/native-http/index.js#L5

```bash
sprkl -- node index.js
```

Then, trigger the server by calling the deployed server API, at any path:

```bash
curl localhost:3000
```
